# Netflix-Movies-Tableau-Dashboard
This project presents an interactive Tableau dashboard analyzing Netflix movie trends, genre performance, revenue contribution, and ROI-based investment insights.

Key Insights Covered :
1. Total Movies, Revenue, Avg Rating & ROI KPIs
2. Movie production trends over time
3. Genre distribution
4. Country-wise production map
5. Budget vs Revenue performance analysis
6. Year-wise popularity
7. Dynamic Top / Bottom movies by Rating or Popularity

Tools Used :
Tableau Public, Excel, Calculated Fields, Parameters

Business Value: 
This dashboard helps evaluate: Content performance, Investment efficiency, Audience engagement, Genre strategy
